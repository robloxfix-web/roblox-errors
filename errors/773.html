Error 773: Teleport — Wrong Place ID

Updated 2026
💡 What Is This Error?

The teleport failed because the destination is a private place, a reserved server, or the place ID used does not allow public access.

⚠️ Common Causes
  • 1 Teleport destination is a private or restricted place
  • 2 Developer used an incorrect place ID in the teleport script
  • 3 You do not have permission to access the destination
✅ How to Fix It — Step by Step
  • 1 Contact the game developer — the teleport script likely has an incorrect place ID
  • 2 If the destination is a reserved server, you need an invitation
  • 3 Try joining the destination game directly from its Roblox page to check if it is accessible
Still not fixed? Contact Roblox official support for further assistance
