Error 805: Unknown Error — Try Again

Updated 2026
💡 What Is This Error?

Roblox encountered an unspecified error. This is a catch-all error code that can appear for various transient issues. It almost always resolves after a retry.

⚠️ Common Causes
  • 1 Temporary network glitch
  • 2 Brief Roblox server hiccup
  • 3 Unknown transient error
✅ How to Fix It — Step by Step
  • 1 Wait 10-30 seconds and try again
  • 2 Restart Roblox completely and retry your action
  • 3 If it persists, check status.roblox.com and report to Roblox Support
