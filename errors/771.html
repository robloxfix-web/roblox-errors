Error 771: Teleport — Server Not Found

Updated 2026
💡 What Is This Error?

The specific server you were being teleported to no longer exists. The server may have shut down between the time the teleport was initiated and when you arrived.

⚠️ Common Causes
  • 1 Target server shut down during the teleport
  • 2 Server reached maximum capacity just before you joined
  • 3 Stale server ID being used in the teleport script
✅ How to Fix It — Step by Step
  • 1 Try the teleport again — a new server should be available
  • 2 If the problem repeats, report it to the game developer
  • 3 Rejoin the game from the game page and try the teleport from the start
Still not fixed? Contact Roblox official support for further assistance
