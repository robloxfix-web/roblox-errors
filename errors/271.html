Error 271: Game Kicked — AFK or Rule

Updated 2026
💡 What Is This Error?

You were kicked from the game either because all players went AFK simultaneously, or because you personally triggered a kick rule set by the game developer.

⚠️ Common Causes
  • 1 All players in the server were idle (AFK) at the same time
  • 2 You broke a server rule that triggers an automatic kick
  • 3 Low player count caused the server to auto-close
✅ How to Fix It — Step by Step
  • 1 Rejoin the game immediately — this kick is usually temporary
  • 2 Stay active in the game to avoid AFK kicks
  • 3 Read the game description for any specific server rules
