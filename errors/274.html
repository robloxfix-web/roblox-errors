Error 274: Server Shut Down by Developer

Updated 2026
💡 What Is This Error?

The developer of the game manually shut down the server you were in, or set the game to private. You cannot rejoin the same server but may be able to join another.

⚠️ Common Causes
  • 1 Developer shut down servers to push a game update
  • 2 Game was set to private by the developer
  • 3 Developer ended an event or temporary server
✅ How to Fix It — Step by Step
  • 1 Wait 10-15 minutes and try joining again from the game page
  • 2 Check the game developer's Twitter or Discord for update announcements
  • 3 Try joining a different server of the same game if available
Still not fixed? Contact Roblox official support for further assistance
