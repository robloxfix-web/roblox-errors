Error 770: Teleport — Game Not Found

Updated 2026
💡 What Is This Error?

The teleport failed because the destination game does not exist or has been deleted. The place ID used in the teleport script does not match any active Roblox game.

⚠️ Common Causes
  • 1 The destination game was deleted by its developer
  • 2 The place ID in the teleport script is incorrect
  • 3 The game is private and you do not have access
✅ How to Fix It — Step by Step
  • 1 The issue is with the game's code — contact the developer
  • 2 Check if the destination game still exists by searching for it on Roblox
  • 3 If you are the developer, verify the place ID in your teleport script is correct
Still not fixed? Contact Roblox official support for further assistance
